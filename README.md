## 📖 Abstract
Breast cancer is one of the most common cancers among women worldwide.  
Early detection plays a crucial role in reducing mortality rates. This project leverages **Machine Learning** to classify tumors as *Benign (non-cancerous)* or *Malignant (cancerous)* based on patient data.  

The system integrates **Python Machine Learning Model** into a **Django Web Application** with **MySQL database**, providing a **full-stack solution** for prediction, dataset visualization, and history tracking.  

The system uses:
- **Data Preprocessing** (handling missing values, scaling features)  
- **Random Forest Classifier** for classification  
- **StandardScaler** for feature normalization  
- **Classification Report** for evaluating model accuracy

## 📊 Dataset Information
The project uses the **Breast Cancer Wisconsin (Diagnostic) Dataset** from UCI/Kaggle.  
It includes 30 input features such as **radius, texture, perimeter, area, smoothness, concavity, symmetry** etc., and the target variable **diagnosis**:  
- **M → Malignant (1)**  
- **B → Benign (0)**  

The system also provides:  
- Dataset Statistics (mean, std, min, max, quartiles)  
- Class Distribution (Benign vs Malignant)  
- Dataset Records (for exploration)
